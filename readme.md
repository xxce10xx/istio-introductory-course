# Curso introductorio de Istio
Este repositorio muestra la configuración principal de Istio, incluyendo DestinationRules, VirtualServices, ServiceEntry y otros recursos relacionados. También se proporciona el stack de observabilidad compuesto por Prometheus, Alloy, Loki, Grafana y Jaeger.

### Observaciones
> Este repositorio representa la entrega final del participante ***Cedric Bardalez Hall*** en el curso _Istio Service Mesh_ de la empresa **Galaxy Training**
